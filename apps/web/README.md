# MYReSolve web application

Next.js App Router workspace for the MYReSolve Platform migration.

## Status

Phase 1 scaffold only. The assessment interface has **not** been ported.

Homepage message:

> MYReSolve Platform v1.0 migration environment

## Stack

- Next.js (App Router)
- TypeScript
- ESLint
- Standard CSS / CSS Modules (no Tailwind)

## Scripts

```bash
npm run dev
npm run lint
npm run build
npm run start
```
